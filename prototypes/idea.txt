3. Modellidee: Low-Budget Embedding statt Training
Du brauchst kein selbst trainiertes Modell für den Anfang. Nutze:

✨ Sentence Transformers (sentence-transformers Library von HuggingFace)
z. B. all-MiniLM-L6-v2 → wandelt Texte in 384-D-Vektoren um
damit kannst du direkt Clustering, Ähnlichkeiten & Visualisierungen machen
Beispiel:

from sentence_transformers import SentenceTransformer
from sklearn.manifold import TSNE
import matplotlib.pyplot as plt

model = SentenceTransformer('all-MiniLM-L6-v2')
embeddings = model.encode(list_of_texts)
tsne = TSNE(n_components=2, perplexity=5)
reduced = tsne.fit_transform(embeddings)

# 2D-Visualisierung
plt.scatter(reduced[:, 0], reduced[:, 1])
for i, label in enumerate(labels):
    plt.text(reduced[i, 0], reduced[i, 1], label)
plt.show()

Ein Netzwerk (Netz) zur Darstellung politischer Bewegungen in deinem Projekt könnte sowohl visuell ansprechend als auch inhaltlich bedeutungsvoll gestaltet sein. Hier ist eine konkrete Vorstellung davon, wie es aussehen und funktionieren könnte:

🕸️ Politisches Bewegungsnetz – visuelle Beschreibung

📌 Knoten (Nodes)
Jeder Knoten repräsentiert eine politische Bewegung oder Partei.
Größe des Knotens: z. B. symbolisiert Reichweite, Präsenz oder Einfluss.
Farbe: ideologische Grundtendenz (z. B. grün = ökologisch, rot = sozialistisch, blau = konservativ).
Stil/Form: kann für künstlerische Interpretationen stehen (z. B. organisch vs. geometrisch, glatt vs. rau).
🔗 Verbindungen (Edges)
Eine Linie zwischen zwei Knoten zeigt eine thematische Nähe, also z. B.:
Gemeinsame Narrative (z. B. Kapitalismuskritik, Umweltschutz)
Überlappende Forderungen
Ähnliche Werte (aus den Embeddings berechnet: hoher Cosine Similarity)
Dicke oder Helligkeit der Linie: wie stark ist die Verbindung?
Transparenz oder gepunktete Linien: hypothetische/indirekte Verbindungen (z. B. „potenzielle Fusion“)
🧭 Layout-Idee
Das Netz kann mithilfe eines Force-Directed Graph Layouts automatisch „in Form gezogen“ werden:
Bewegungen mit hoher Ähnlichkeit ziehen sich an
große Unterschiede stoßen ab
ergibt eine organische Anordnung, die ideologische Cluster sichtbar macht
💡 Interaktive Features (optional)
Hover/Click: Zeigt Infos zur Bewegung, Zitate, Kernforderungen
Filter: z. B. nach Thema (Umwelt, Migration, Ökonomie)
"Was wäre wenn?"-Modus: Bewegung A + B kombinieren → GPT generiert Synthese-Programm
Bewegungen selbst zeichnen (für Besucher): DIY-Fusion oder kritische Perspektive
🎨 Künstlerische Erweiterungsideen
Visualisiere Spannungen durch vibrierende Linien oder visuelle „Risse“
Resonanzen durch pulsierende Verbindungen (z. B. wenn viele Gemeinsamkeiten bestehen)
Bewegung über Zeit: kleine Animationen, wie sich Bewegungen annähern oder entfernen
Verwendung typografischer Texturen oder visueller Metaphern (z. B. Baum, Nervensystem, Magnetfelder)
🔧 Tools für Umsetzung
p5.js oder Three.js für künstlerische/animierte Netzwerke (WebGL)
D3.js für strukturierte, interaktive Graphen im Web
Gephi für statische Netzdiagramme mit Export (gut für Prototypen)
Python (Plotly/NetworkX) für einfache Visualisierung direkt aus Embeddings
